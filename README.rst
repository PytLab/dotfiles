========
Dotfiles
========

**Introduction**

A script to setup my configuration files(`.vimrc`, `.vim`, `.pip etc`)

**Quick install & uninstall**

install::

    $ ./setup --install

uninstall::

    $ ./setup --uninstall

install offline (with `bundle.tar.gz` supplied)::

    $ ./setup --install --offline
